# Store Listing and Release Metadata Checklist

## Required store assets
- app icon
- feature graphic
- phone screenshots:
  - dashboard
  - debt detail
  - strategy simulator
  - scan/import review
  - backups/privacy settings
- tablet screenshots if tablet support is claimed

## Metadata checklist
- short description
- full description
- release notes / what's new
- support email
- privacy policy URL
- data safety answers
- content rating answers
- subscription disclosure copy for monthly and yearly plans

## Billing listing notes
- Product id: `premium`
- Base plans:
  - `monthly`
  - `yearly`
- Verify plan labels, pricing copy, and renewal wording in Play Console before rollout.

## Final pre-publish checks
- package name matches Play Console app
- signing key / app signing enrollment confirmed
- closed test track has at least one validated artifact
- screenshots and descriptions match the current UI and supported features
